## 开发

本项目使用[Bun](https://bun.sh/)构建。

```bash
bun i
bun run build
# 🧹  Cleaned up dist/user-script-0.0.1 directory.
# 🔥  Built user-script.
# 📦  Bundled     => dist/user-script-0.0.1/index.js
# 📦  Compressed  => dist/user-script-0.0.1.zip
```

将`dist/user-script-0.0.1/index.js`文件添加到 Tampermonkey 中。
